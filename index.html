Hubungkan ke Supabase agar data tersimpan permanen dan bisa diakses dari perangkat manapun.

1Buat Project Supabase

Buka supabase.com → New project → pilih region Southeast Asia (Singapore) → tunggu selesai (~2 menit).

2Jalankan schema.sql

Di Supabase Dashboard → SQL Editor → New query → paste seluruh isi file schema.sql → klik Run.

3Masukkan Kredensial

Temukan di Supabase Dashboard → Project Settings → API
